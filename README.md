# Vuex Todo Manager

> This project is Todo app front using Vue.js and Vuex statemanagement with jsonplaceholder API

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your tests

```
npm run test
```

### Lints and fixes files

```
npm run lint
```
